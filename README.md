# Strategic Focus Items Board

A standalone, no-auth web app for managing strategic focus items with Don and Sean.

**Features:**
- Add items with: description, expected outcome, lens (Revenue/Cost/Efficiency), owner, status, target date
- Edit status and dates on the fly
- Real-time browser storage (localStorage)
- Ignition Cost app styling

**Deploy to Vercel (1 click):**
1. Go to https://vercel.com/import
2. Import this repo
3. Deploy (takes ~30 seconds)

**Deploy to GitHub Pages:**
1. Push to GitHub
2. Enable Pages in repo settings → main branch
3. Live at https://yourusername.github.io/strategic-focus-app

**Or use locally:**
```bash
python3 -m http.server 8000
# Visit http://localhost:8000
```
